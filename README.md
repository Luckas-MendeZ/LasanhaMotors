## Sobre o projeto

O **LasanhaMotors** é um projeto full stack desenvolvido para praticar conceitos de desenvolvimento web moderno.

Apesar do tema ser totalmente inspirado na cultura automotiva brasileira e em memes do mundo das "lasanhas", toda a arquitetura, organização do código e funcionalidades foram desenvolvidas como em uma aplicação real.

O objetivo foi unir minhas duas paixões: **programação** e **carros modificados**.

## Objetivo

Este projeto não possui qualquer finalidade comercial.

Ele foi criado exclusivamente para:

* praticar desenvolvimento Full Stack;
* aprender arquitetura de aplicações;
* desenvolver APIs REST;
* praticar consumo de APIs;
* melhorar organização de código;
* construir um portfólio que seja técnico, mas também reflita minha personalidade.
